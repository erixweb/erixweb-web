Proyectos

Reporeq

Reporeq es un proyecto en el que el usuario proporciona el nombre de usuario de alguien en Github y entonces, se hace un fetch a la API de Github devolviendo las repos y información publica del usuario.

Astrowiki

Encuentra solución a todas tus pregunta sobre Astro.js. Un framework de JavaScript hecho para hacer páginas web rápidas con el JavaScript necesario.

Windful

Windful es una librería de CSS basada en Tailwind. Está compuesta por 300 lineas, muy ligera pero eficaz.

WetSpain

WetSpain es una página para ver el clima en diferentes ciudades de España.

ToDo List

ToDo es un programa para hacer una lista de tus tareas ordenadamente.

Week-Recap

Week-Recap es una aplicación web que he hecha por ErixWeb para hacer una lista de tus gastos semanales.

BinAPI

BinAPI es una API con API calls unlimitadas en la que, puedes traducir binario.

LSXDom

LSX es un clone de JSX hecho con JavaScript Vanilla e menos de 2.5kB.
