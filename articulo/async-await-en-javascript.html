Async/await en JavaScript

En este blog te enseñaré a cómo usar y para que sirve async y await en JavaScript. Además te explicare por qué esto no se hace en Python o otros lenguajes de programación.

¿Qué es async y await?

Async y await en pocas palabras, hacen las promesas de JavaScript mas fáciles. Async hace que una función devuelva una promesa y await espera una promesa. Mas abajo veras que es todo esto que estoy explicando.

¿Cómo usar async y await?

Para usar async es mucho mas fácil de lo que estas pensando. Sería tan simple como añadir la palabra async antes de una función. 


async function devolver(text) {
    return text
}

El código de arriba es lo mismo que esto: 


function devolver(text) {
    Promise.resolve(text)
}

En estos dos casos la función devuelve una promesa.

¿Cómo puedo usar await?

Await puede parecer una vez mas, complejo. Pero es tan fácil como el async.

Algo importante a saber del await es que solo puede usarse dentro de una función asincrona, es decir, la función tiene que tener async. Lo que hace await es esperar a una promesa. 


async function devolver(text) {
    let promesa = new Promise(function(resolve, reject) {
        resolve(text);
    })
    document.querySelector("body").innerHTML = await promesa
}
